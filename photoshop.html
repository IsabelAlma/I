Photoshop for UX Design

Background

Following the LinkedIn’s Learning Path “Become a User Experience Designer “, one of the courses “Photoshop for UX Design” centered around the wildly used program, Photoshop. Here, I learned how to create wireframes, flowcharts, and mockups using the tool, furthering my knowledge on how a User Experience Designer would go about visualizing ideas.

Role

Methods & Tools

Practices

Outcome

The course went over several features and tools Photoshop has to offer for when approaching it from a UX standpoint. While I personally have used the program before, the thought of using it to create a wireframe did not cross my mind. Photoshop was taught to me as a digital media platform, and was used as a poster making, photo editing, image cropping platform! Now I have knowledge in setting up an artboard, annotating my wireframe, and extracting CSS values from a design.
